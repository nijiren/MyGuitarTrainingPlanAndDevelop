# ギタートレーニングプラン
## Matt Schofieldスタイル習得への道

### 対象者プロフィール
- **ギター暦**: 15年以上
- **技術レベル**: 中級〜上級（Pops〜HR/HM対応可能、Angra/Dream Theaterレベル）
- **得意分野**: バッキング、カッティング、高速ダウンストローク
- **課題**: 音楽理論（コードトーン、トライアド）、インプロヴァイズフレージング
- **目標**: Matt Schofieldスタイルのブルース/ジャズフレージング習得

---

## トレーニングの全体像

### 3つの柱
1. **音楽理論の実践的理解**（コードトーン、トライアド、スケール）
2. **コードトーンを意識したフレージング練習**
3. **インプロヴァイズ能力の向上**

### 期間目安
- **基礎固めフェーズ**: 3-4ヶ月
- **応用フェーズ**: 3-4ヶ月
- **習熟フェーズ**: 継続的

## フェーズ1: 基礎固め（3-4ヶ月）

### 週1-2: コードトーンの理解と指板の把握

#### 目標
- コードトーン（ルート、3rd、5th、7th）を指板上で視覚的に理解する
- トライアドの3つの音を正確に把握する

#### 練習内容

**1. トライアドの指板把握（毎日15-20分）**
- **メジャートライアド**（ルート、3rd、5th）を全キーで練習
  - Cメジャーから始めて、半音ずつ上げていく
  - 各ポジションで3つの音を正確に把握
  - ルートポジション、第1転回形、第2転回形を練習
  
- **マイナートライアド**（ルート、♭3rd、5th）
  - 同様に全キーで練習
  
- **練習方法**:
  ```
  例: Cメジャートライアド
  - ルートポジション: C(ルート) → E(3rd) → G(5th)
  - 第1転回形: E(3rd) → G(5th) → C(ルート)
  - 第2転回形: G(5th) → C(ルート) → E(3rd)
  ```

**2. コードトーンの視覚化（毎日10分）**
- 指板図を見ながら、各コードの構成音を確認
- コード進行に合わせて、各コードのトライアドを弾く
  - 例: Am → Dm → G → C の進行で、各コードのトライアドを弾く

**3. ブルース進行でのトライアド練習（毎日15分）**
- 12小節ブルース進行（I-IV-V）で練習
  - 例: Aブルース（A7 → D7 → A7 → A7 → D7 → D7 → A7 → A7 → E7 → D7 → A7 → E7）
  - 各コードのトライアドを弾く
  - 最初はゆっくり、徐々にテンポアップ

#### チェックポイント
- [ ] 全キーでメジャー/マイナートライアドが指板上で即座に分かる
- [ ] コード進行を見ただけで、各コードのトライアドが弾ける

---

### 週3-4: セブンスコードとコードトーンの拡張

#### 目標
- セブンスコード（7th、m7、maj7、m7♭5）の構成音を理解
- コードトーンを基盤としたフレージングの基礎

#### 練習内容

**1. セブンスコードの構成音把握（毎日15分）**
- **ドミナント7th**（1, 3, 5, ♭7）
  - 例: A7 = A, C#, E, G
- **マイナー7th**（1, ♭3, 5, ♭7）
  - 例: Am7 = A, C, E, G
- **メジャー7th**（1, 3, 5, 7）
  - 例: Amaj7 = A, C#, E, G#
- **マイナー7th♭5**（1, ♭3, ♭5, ♭7）
  - 例: Am7♭5 = A, C, E♭, G

**2. コードトーンアルペジオ練習（毎日20分）**
- 各セブンスコードのアルペジオを全ポジションで練習
- ブルース進行でコードトーンアルペジオを弾く
  - 例: A7 → D7 → A7 → A7... の進行で、各コードのアルペジオ

**3. コードトーンを意識した簡単なフレーズ（毎日15分）**
- コードトーンを中心に、少し装飾音を加えたフレーズ
- 例: A7のコードトーン（A, C#, E, G）を中心に、ペンタトニックの音を少し加える

#### チェックポイント
- [ ] 主要なセブンスコードの構成音が即座に分かる
- [ ] コード進行に合わせて、コードトーンアルペジオがスムーズに弾ける

---

### 週5-8: スケールとコードトーンの関係

#### 目標
- スケールとコードトーンの関係を理解
- コードトーンを基盤としたスケール練習

#### 練習内容

**1. メジャースケールとコードトーンの関係（毎日15分）**
- メジャースケールの各音がコードトーンになることを理解
  - 例: Cメジャースケール（C, D, E, F, G, A, B）
  - Cメジャー = C, E, G（1, 3, 5）
  - Dm = D, F, A（2, 4, 6）
  - Em = E, G, B（3, 5, 7）
  - など

**2. ブルーススケールとコードトーンの融合（毎日20分）**
- ブルーススケール（ペンタトニック + ♭5）を弾きながら、コードトーンを意識
- コードトーンを「着地点」として使う練習
  - 例: A7のコードトーン（A, C#, E, G）を強調しながら、ブルーススケールでフレーズを構築

**3. ドリアンスケールの習得（毎日15分）**
- マイナー7thコードで使えるドリアンスケール
  - 例: Am7 → Aドリアンスケール（A, B, C, D, E, F#, G）
  - コードトーン（A, C, E, G）を含むスケール

**4. ミクソリディアンスケールの習得（毎日15分）**
- ドミナント7thコードで使えるミクソリディアンスケール
  - 例: A7 → Aミクソリディアン（A, B, C#, D, E, F#, G）
  - コードトーン（A, C#, E, G）を含むスケール

#### チェックポイント
- [ ] コード進行を見て、適切なスケールが選べる
- [ ] コードトーンを意識しながら、スケールでフレーズが作れる

---

### 週9-12: コードトーンを意識したフレージング

#### 目標
- コードトーンを「着地点」として使うフレージング
- Matt Schofieldスタイルの基礎フレージング

#### 練習内容

**1. コードトーン着地練習（毎日20分）**
- ブルース進行で、各コードのコードトーンに着地するフレーズ
- 例: A7の小節では、A, C#, E, Gのいずれかに着地
- 最初は1小節1音から、徐々に複雑に

**2. コードトーンを中心としたフレーズ構築（毎日25分）**
- コードトーンを「骨格」として、装飾音を加える
- 例: A7のコードトーン（A, C#, E, G）を中心に、ブルーススケールやペンタトニックの音を加える
- リズムバリエーションも練習

**3. トライアドを横断するフレーズ（毎日20分）**
- コード進行に合わせて、各コードのトライアドを横断するフレーズ
- 例: Am → Dm → G → C の進行で、各トライアドを滑らかに繋ぐ

**4. Matt Schofieldのフレーズ分析（毎日15分）**
- 実際のMatt Schofieldのソロを分析
- コードトーンがどこで使われているか確認
- フレーズをコピーして、コードトーンの使い方を学ぶ

#### チェックポイント
- [ ] コード進行に合わせて、コードトーンを意識したフレーズが作れる
- [ ] コードトーンを「着地点」として自然に使える

## フェーズ2: 応用（3-4ヶ月）

### 週13-16: ジャズブルースの理解と実践

#### 目標
- ジャズブルース進行の理解
- より複雑なコード進行でのコードトーン活用

#### 練習内容

**1. ジャズブルース進行の習得（毎日20分）**
- 標準的なジャズブルース進行
  - 例: Am7 → Dm7 → Am7 → Am7 → Dm7 → Dm7 → Am7 → Am7 → E7 → Dm7 → Am7 → E7
- 各コードのコードトーンを把握

**2. ツー・ファイブ・ワン進行の練習（毎日25分）**
- ジャズの基本進行（ii-V-I）
- 例: Am7 → D7 → Gmaj7
- 各コードのコードトーンを意識したフレージング
- スケールチェンジの練習

**3. コードトーンを基盤としたインプロヴァイズ（毎日30分）**
- バッキングトラックに合わせて、コードトーンを意識したインプロヴァイズ
- 最初はゆっくり、コードトーンを明確に
- 徐々にスピードアップ、装飾音を加える

**4. リズムのバリエーション（毎日15分）**
- シンコペーション、タイ、トリプレットなど
- コードトーンをリズミックに配置

#### チェックポイント
- [ ] ジャズブルース進行で、コードトーンを意識したフレーズが作れる
- [ ] ツー・ファイブ・ワン進行で、スムーズにスケールチェンジできる

### 週17-20: 高度なコードトーン活用

#### 目標
- テンション（9th、11th、13th）の理解
- より色彩豊かなフレージング

#### 練習内容

**1. テンションの理解（毎日15分）**
- 9th、11th、13thの理解
- コードトーン + テンションの組み合わせ
- 例: Am7(9) = A, C, E, G, B

**2. コードトーン + テンションのフレージング（毎日25分）**
- コードトーンを基盤に、テンションを加えたフレーズ
- よりジャズ的な響きを目指す

**3. ポリコードの理解（毎日20分）**
- 複数のコードが同時に鳴る響き
- コードトーンを意識した、より複雑なフレージング

**4. Matt Schofieldスタイルの深掘り（毎日30分）**
- より多くのMatt Schofieldのソロを分析・コピー
- コードトーンとテンションの使い方を学ぶ
- 自分のフレージングに取り入れる

#### チェックポイント
- [ ] テンションを理解し、フレージングに活用できる
- [ ] Matt Schofieldスタイルのフレージングが再現できる

### 週21-24: インプロヴァイズ能力の向上

#### 目標
- 自由にインプロヴァイズできる
- コードトーンを自然に意識したフレージング

#### 練習内容

**1. 長時間インプロヴァイズ（毎日30分）**
- バッキングトラックに合わせて、10-15分間連続でインプロヴァイズ
- コードトーンを意識しつつ、自由にフレーズを展開
- 繰り返しを避け、常に新しいフレーズを探す

**2. 異なるキーでの練習（毎日20分）**
- 全12キーで同じ進行を練習
- コードトーンの把握を全キーで確実に

**3. リズムの複雑化（毎日20分）**
- 複雑なリズムパターンでのコードトーン活用
- シンコペーション、ポリリズムなど

**4. オリジナルフレーズの構築（毎日30分）**
- コードトーンを基盤とした、オリジナルフレーズの開発
- Matt Schofieldスタイルを参考にしつつ、自分の個性を出す

#### チェックポイント
- [ ] 任意のコード進行で、コードトーンを意識したインプロヴァイズができる
- [ ] 長時間インプロヴァイズしても、フレーズが枯渇しない

## フェーズ3: 習熟（継続的）

### 継続的な練習内容

#### 毎日のルーティン（1-2時間）

**1. ウォームアップ（10分）**
- トライアドの確認練習
- コードトーンアルペジオ

**2. スケール練習（15分）**
- メジャー、マイナー、ドリアン、ミクソリディアン
- コードトーンを意識しながら

**3. コード進行練習（20分）**
- ブルース、ジャズブルース、ツー・ファイブ・ワン
- コードトーンを意識したフレージング

**4. インプロヴァイズ（30分）**
- バッキングトラックに合わせて
- コードトーンを自然に意識

**5. コピー・分析（15分）**
- Matt Schofield、John Mayer、Cory Wongなどのソロ分析
- コードトーンの使い方を学ぶ

#### 週次タスク

**1. 新しいフレーズの開発（週1回、1時間）**
- オリジナルフレーズの構築
- 録音して、後で分析

**2. ライブセッション（可能であれば）**
- 他のミュージシャンとのセッション
- 実践的なインプロヴァイズ

**3. 録音・分析（週1回、30分）**
- 自分のインプロヴァイズを録音
- コードトーンが適切に使えているか分析
- 改善点を見つける

## 具体的な練習例

### 練習例1: トライアドの指板把握

```
Cメジャートライアド（C, E, G）を全ポジションで練習

ポジション1（開放弦付近）:
e|--0--3--
B|--1-----
G|--0-----
D|--2-----
A|--3-----
E|--------

ポジション2（3フレット付近）:
e|--3--7--
B|--5-----
G|--4-----
D|--5-----
A|--3-----
E|--------

...（全ポジションで同様に）
```

### 練習例2: ブルース進行でのコードトーン着地

```
Aブルース進行（A7 → D7 → A7 → A7...）

A7の小節:
- コードトーン（A, C#, E, G）のいずれかに着地
- 例: ブルーススケールでフレーズを展開し、最後にA（ルート）に着地

D7の小節:
- コードトーン（D, F#, A, C）のいずれかに着地
- 例: フレーズを展開し、最後にD（ルート）に着地
```

### 練習例3: コードトーンを中心としたフレーズ

```
A7のコードトーン（A, C#, E, G）を中心に、装飾音を加える

例:
e|-----------------3-5-3--------
B|-------------3-5---------------
G|---------2-4------------------
D|-----2-4----------------------
A|-3-5--------------------------
E|------------------------------

（A, C#, E, Gを中心に、ペンタトニックの音を加える）
```
