# ギタートレーニングプロジェクト

## プロジェクト概要

Matt Schofieldスタイルのブルース/ジャズフレージング習得を目標とした、ギタートレーニングプランと補助アプリの開発プロジェクトです。

## ディレクトリ構造

```
ギタートレーニング/
├── 00_概要/              # プロジェクト全体の概要
├── 01_トレーニングプラン/  # ギタートレーニングの詳細プラン
│   ├── フェーズ1_基礎固め/
│   ├── フェーズ2_応用/
│   └── フェーズ3_習熟/
├── 02_アプリ開発/        # アプリ開発の計画とロードマップ
│   ├── 計画書/
│   ├── フェーズ0_準備設計/
│   ├── フェーズ1_MVP開発/
│   ├── フェーズ2_コア機能拡張/
│   └── フェーズ3_モバイルアプリ/
├── 03_技術仕様/          # 技術的な詳細仕様
└── 04_参考資料/          # 参考資料・リソース
```

## 対象者プロフィール

- **ギター暦**: 15年以上
- **技術レベル**: 中級〜上級（Pops〜HR/HM対応可能、Angra/Dream Theaterレベル）
- **得意分野**: バッキング、カッティング、高速ダウンストローク
- **課題**: 音楽理論（コードトーン、トライアド）、インプロヴァイズフレージング
- **目標**: Matt Schofieldスタイルのブルース/ジャズフレージング習得

## 3つの柱

1. **音楽理論の実践的理解**（コードトーン、トライアド、スケール）
2. **コードトーンを意識したフレージング練習**
3. **インプロヴァイズ能力の向上**

## 期間目安

- **基礎固めフェーズ**: 3-4ヶ月
- **応用フェーズ**: 3-4ヶ月
- **習熟フェーズ**: 継続的
