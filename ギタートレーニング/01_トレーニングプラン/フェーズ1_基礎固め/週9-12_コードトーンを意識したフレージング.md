# 週9-12: コードトーンを意識したフレージング

## 目標
- コードトーンを「着地点」として使うフレージング
- Matt Schofieldスタイルの基礎フレージング

## 練習内容

### 1. コードトーン着地練習（毎日20分）
- ブルース進行で、各コードのコードトーンに着地するフレーズ
- 例: A7の小節では、A, C#, E, Gのいずれかに着地
- 最初は1小節1音から、徐々に複雑に

### 2. コードトーンを中心としたフレーズ構築（毎日25分）
- コードトーンを「骨格」として、装飾音を加える
- 例: A7のコードトーン（A, C#, E, G）を中心に、ブルーススケールやペンタトニックの音を加える
- リズムバリエーションも練習

### 3. トライアドを横断するフレーズ（毎日20分）
- コード進行に合わせて、各コードのトライアドを横断するフレーズ
- 例: Am → Dm → G → C の進行で、各トライアドを滑らかに繋ぐ

### 4. Matt Schofieldのフレーズ分析（毎日15分）
- 実際のMatt Schofieldのソロを分析
- コードトーンがどこで使われているか確認
- フレーズをコピーして、コードトーンの使い方を学ぶ

## チェックポイント
- [ ] コード進行に合わせて、コードトーンを意識したフレーズが作れる
- [ ] コードトーンを「着地点」として自然に使える

## 具体的な練習例

### 練習例2: ブルース進行でのコードトーン着地

```
Aブルース進行（A7 → D7 → A7 → A7...）

A7の小節:
- コードトーン（A, C#, E, G）のいずれかに着地
- 例: ブルーススケールでフレーズを展開し、最後にA（ルート）に着地

D7の小節:
- コードトーン（D, F#, A, C）のいずれかに着地
- 例: フレーズを展開し、最後にD（ルート）に着地
```

### 練習例3: コードトーンを中心としたフレーズ

```
A7のコードトーン（A, C#, E, G）を中心に、装飾音を加える

例:
e|-----------------3-5-3--------
B|-------------3-5---------------
G|---------2-4------------------
D|-----2-4----------------------
A|-3-5--------------------------
E|------------------------------

（A, C#, E, Gを中心に、ペンタトニックの音を加える）
```
