# フェーズ1: 基礎固め（3-4ヶ月）

## 目標

コードトーンとトライアドの理解を深め、指板上で視覚的に把握できるようになる

## 週次プラン

- [週1-2: コードトーンの理解と指板の把握](週1-2_コードトーンの理解と指板の把握.md)
- [週3-4: セブンスコードとコードトーンの拡張](週3-4_セブンスコードとコードトーンの拡張.md)
- [週5-8: スケールとコードトーンの関係](週5-8_スケールとコードトーンの関係.md)
- [週9-12: コードトーンを意識したフレージング](週9-12_コードトーンを意識したフレージング.md)

## チェックポイント

### 1ヶ月目
- [ ] 全キーでメジャー/マイナートライアドが指板上で即座に分かる
- [ ] コード進行を見ただけで、各コードのトライアドが弾ける

### 2ヶ月目
- [ ] 主要なセブンスコードの構成音が即座に分かる
- [ ] コード進行に合わせて、コードトーンアルペジオがスムーズに弾ける

### 3ヶ月目
- [ ] コード進行を見て、適切なスケールが選べる
- [ ] コードトーンを意識しながら、スケールでフレーズが作れる

### 4ヶ月目
- [ ] コード進行に合わせて、コードトーンを意識したフレーズが作れる
- [ ] コードトーンを「着地点」として自然に使える
