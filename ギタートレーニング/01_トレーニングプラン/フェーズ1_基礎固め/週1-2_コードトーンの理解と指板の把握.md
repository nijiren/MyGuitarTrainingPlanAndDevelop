# 週1-2: コードトーンの理解と指板の把握

## 目標
- コードトーン（ルート、3rd、5th、7th）を指板上で視覚的に理解する
- トライアドの3つの音を正確に把握する

## 優先すべきポジション（Matt Schofieldスタイル向け）

### フェーズ1: 優先ポジション（週1-2で重点的に）

**5フレット付近（ミドルレンジ）を主軸に**
- **5弦ルート（Aポジション）**: 5-8フレット付近
  - ブルース/ジャズで最も頻繁に使用されるポジション
  - A、D、G、Cなどのコードで多用
  - 例: A7のルートが5弦5フレット
  
- **6弦ルート（Eポジション）**: 5-8フレット付近
  - E、A、Dなどのコードで重要
  - 例: E7のルートが6弦5フレット

**理由**:
- Matt Schofieldスタイルでは、ミドルレンジ（5-12フレット）でのフレージングが中心
- 5フレット付近は、コードトーンを意識したフレージングに最適
- ブルース/ジャズでよく使われるキー（A、E、D、G）に適している

### フェーズ2: 拡張ポジション（週3-4以降）

**12フレット付近（オクターブ上）**
- 5弦ルート、6弦ルートのオクターブ上
- より高音域でのフレージングに必要

**開放弦付近（0-3フレット）**
- 低音域でのコードトーン把握
- 特にA、E、Dなどのコードで有効

### 練習の進め方

1. **週1-2**: 5フレット付近の5弦・6弦ルートに集中
2. **週3-4**: 12フレット付近を追加
3. **週5-8以降**: その他のポジションを段階的に追加

## 練習内容

### 1. トライアドの指板把握（毎日15-20分）

#### 優先ポジションでの練習（週1-2の重点）

**5弦ルート（Aポジション）を主軸に**
- **A、D、G、C**のメジャートライアドを5フレット付近で練習
- 例: Aメジャートライアド（5弦5フレット = A）
  ```
  5弦ルートポジション（5-8フレット）:
  e|--5--8--
  B|--5-----
  G|--6-----
  D|--7-----
  A|--5----- (ルート)
  E|--------
  ```

**6弦ルート（Eポジション）を主軸に**
- **E、A、D**のメジャートライアドを5フレット付近で練習
- 例: Eメジャートライアド（6弦5フレット = E）
  ```
  6弦ルートポジション（5-8フレット）:
  e|--5-----
  B|--5-----
  G|--4-----
  D|--6-----
  A|--7-----
  E|--5----- (ルート)
  ```

#### メジャートライアド（ルート、3rd、5th）
- **優先キー**: A、E、D、G、Cから始める（ブルース/ジャズで多用）
- 各ポジションで3つの音を正確に把握
- ルートポジション、第1転回形、第2転回形を練習
- **週1-2は5フレット付近に集中**、他のポジションは後回し

#### マイナートライアド（ルート、♭3rd、5th）
- **優先キー**: Am、Em、Dm、Gm、Cmから始める
- 同様に5フレット付近を主軸に練習

#### 練習方法
```
例: Cメジャートライアド
- ルートポジション: C(ルート) → E(3rd) → G(5th)
- 第1転回形: E(3rd) → G(5th) → C(ルート)
- 第2転回形: G(5th) → C(ルート) → E(3rd)
```

### 2. コードトーンの視覚化（毎日10分）
- 指板図を見ながら、各コードの構成音を確認
- コード進行に合わせて、各コードのトライアドを弾く
  - 例: Am → Dm → G → C の進行で、各コードのトライアドを弾く

### 3. ブルース進行でのトライアド練習（毎日15分）
- 12小節ブルース進行（I-IV-V）で練習
  - 例: Aブルース（A7 → D7 → A7 → A7 → D7 → D7 → A7 → A7 → E7 → D7 → A7 → E7）
  - 各コードのトライアドを弾く
  - 最初はゆっくり、徐々にテンポアップ

## チェックポイント
- [ ] 全キーでメジャー/マイナートライアドが指板上で即座に分かる
- [ ] コード進行を見ただけで、各コードのトライアドが弾ける

## 具体的な練習例

### 練習例1: トライアドの指板把握（優先ポジション）

#### Aメジャートライアド（A, C#, E）- 5弦ルート

**5弦ルートポジション（5-8フレット）** - **優先**
```
e|--5--8--  (E, A)
B|--5--9--  (C#, E)
G|--6--9--  (E, A)
D|--7--9--  (A, C#)
A|--5--7--  (A, E) ← ルート
E|--------
```

#### Eメジャートライアド（E, G#, B）- 6弦ルート

**6弦ルートポジション（5-8フレット）** - **優先**
```
e|--5--8--  (E, G#)
B|--5--7--  (G#, B)
G|--4--7--  (B, E)
D|--6--9--  (E, G#)
A|--7--9--  (G#, B)
E|--5--7--  (E, B) ← ルート
```

#### Dメジャートライアド（D, F#, A）- 5弦ルート

**5弦ルートポジション（5-8フレット）** - **優先**
```
e|--5--8--  (A, D)
B|--6--10-- (F#, A)
G|--7--9--  (A, D)
D|--5--7--  (D, A) ← ルート
A|--5--7--  (D, A)
E|--------
```

### 練習の進め方

1. **週1-2**: 上記の3つのポジション（5弦・6弦ルート、5フレット付近）に集中
2. **週3-4**: 12フレット付近（オクターブ上）を追加
3. **週5-8以降**: その他のポジションを段階的に追加

### なぜこのポジションが重要か

- **Matt Schofieldスタイル**: ミドルレンジ（5-12フレット）でのフレージングが中心
- **ブルース/ジャズ**: A、E、D、Gなどのキーが多用される
- **コードトーン着地**: 5フレット付近は、コードトーンを意識したフレージングに最適なポジション
- **実用性**: このポジションをマスターすれば、多くのブルース/ジャズフレーズに対応可能
