# ギタートレーニングプラン
## Matt Schofieldスタイル習得への道

### 対象者プロフィール
- **ギター暦**: 15年以上
- **技術レベル**: 中級〜上級（Pops〜HR/HM対応可能、Angra/Dream Theaterレベル）
- **得意分野**: バッキング、カッティング、高速ダウンストローク
- **課題**: 音楽理論（コードトーン、トライアド）、インプロヴァイズフレージング
- **目標**: Matt Schofieldスタイルのブルース/ジャズフレージング習得

---

## トレーニングの全体像

### 3つの柱
1. **音楽理論の実践的理解**（コードトーン、トライアド、スケール）
2. **コードトーンを意識したフレージング練習**
3. **インプロヴァイズ能力の向上**

### 期間目安
- **基礎固めフェーズ**: 3-4ヶ月
- **応用フェーズ**: 3-4ヶ月
- **習熟フェーズ**: 継続的

## 進捗の確認方法

### 月次チェックリスト

**1ヶ月目**
- [ ] 全キーでメジャー/マイナートライアドが弾ける
- [ ] ブルース進行で、各コードのトライアドが弾ける

**2ヶ月目**
- [ ] セブンスコードの構成音が理解できている
- [ ] コードトーンアルペジオがスムーズに弾ける

**3ヶ月目**
- [ ] コード進行に合わせて、適切なスケールが選べる
- [ ] コードトーンを意識した簡単なフレーズが作れる

**4ヶ月目**
- [ ] コードトーンを「着地点」として使える
- [ ] 簡単なインプロヴァイズができる

**6ヶ月目**
- [ ] ジャズブルース進行で、コードトーンを意識したフレーズが作れる
- [ ] ツー・ファイ五・ワン進行で、スムーズにスケールチェンジできる

**8ヶ月目**
- [ ] テンションを理解し、フレージングに活用できる
- [ ] Matt Schofieldスタイルのフレージングが再現できる

**12ヶ月目**
- [ ] 任意のコード進行で、コードトーンを意識したインプロヴァイズができる
- [ ] 長時間インプロヴァイズしても、フレーズが枯渇しない

## モチベーション維持のコツ

1. **録音して進歩を確認**: 月1回、自分の演奏を録音して、進歩を確認
2. **小さな目標を設定**: 週単位、月単位で小さな目標を設定
3. **好きなアーティストの分析**: Matt Schofield、John Mayerなどのソロを分析して、モチベーションを維持
4. **他のミュージシャンとの交流**: 可能であれば、他のギタリストとセッション
5. **楽しむ**: 理論を学びながらも、音楽を楽しむことを忘れない
