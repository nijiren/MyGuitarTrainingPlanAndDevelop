## マイルストーン

### マイルストーン1: MVPリリース（3-4ヶ月後）
- 基本的なレッスン機能
- メトロノーム・チューナー
- 練習ログ
- Webアプリのみ

### マイルストーン2: コア機能完成（6-7ヶ月後）
- 音楽理論ツール
- タブ譜システム
- バッキングトラック
- ソングライブラリ

### マイルストーン3: モバイルリリース（9-10ヶ月後）
- iOS/Androidアプリ
- オフライン機能
- ネイティブチューナー

### マイルストーン4: 高度な機能（12ヶ月+）
- AI機能
- コミュニティ機能
